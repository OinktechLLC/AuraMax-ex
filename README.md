# AuraMAX — Защита для МАКС Национального Мессенджера

**Простое и мощное браузерное расширение** для блокировки трекинга, cookies, рекламы и рекомендаций на `web.max.ru`.

## Возможности
- Полная блокировка cookies и трекинг-запросов к серверам MAX
- Блокировка рекламы, баннеров, рекомендаций и "каналов"
- Скрытие лишнего интерфейса (боковые панели, промо)
- Защита ваших личных данных

## Установка
1. Скачайте ZIP-архив
2. Распакуйте в папку
3. Откройте `chrome://extensions/` (или `edge://extensions/`)
4. Включите **Режим разработчика**
5. Нажмите **"Загрузить распакованное расширение"** и выберите папку `AuraMAX`

## Структура
- `manifest.json` — настройки
- `content.js` — основной скрипт (блокировка + скрытие элементов)
- `background.js` — сервис-воркер
- `rules.json` — правила declarativeNetRequest
